    Klasifikasi K-NN(K-Nearest Neighbor) dengan Iris- Dataset¶

    Pengertian K-NN(K-Nearest Neighbor)¶

    suatu metode klasifikasi terhadap suatu objek berdasarkan data training yang jaraknya paling dekat dengan objek tersebut. KNN merupakan algoritma supervised learning yakni hasil dari query instance yang baru akan diklasifikasikan berdasarkan mayoritas dari kategori pada algoritma KNN. Dimana kelas yang paling banyak muncul akan menjadi kelas hasil dari klasifikasi.

    Secara umum untuk mendefinisikan jarak antara dua objek x dan y,digunakan rumus Euclidean distance

    dengan :

    dist(x,y) adalah jarak antara dua objek

    Langkah -langkah dalam menggunakan K-NN¶

    Menentukan parameter dari jumlah tetangga yang paling dekat dengan objek(menentukan K)

    Menghitung jarak terhadap masing-masing objek dengan rumus euclidean distance

    Mengurutkan hasil perhitungan euclidean distance

    Menentukan tetangga terdekat berdasarkan jarak minimum ke K

    Menentukan kategori dari tetangga terdekat dengan objek/data

    Gunakan kategori mayoritas sebagai klasifikasi objek/data

    Kelebihan dan kekurangan¶

    ​ Kelebihan :

    1. Lebih efektif di data yang besar

    2. Dapat menghasilkan data yang akurat

    Kekurangan :

    1. Harus menentukan k yang optimal sehingga diperoleh accurasi yang tinggi

    2. Biaya komputasi yang tinggi karena perhitungan jarak harus selalu dilakukan pada setiap

    query instance

    Contoh soal perhitungan K-NN¶

    Diberikan data training berupa 2 atribut yakni Bad dan Good untuk mengklasifikasikan sebuah data apakah termasuk ke dalam Bad atau Good

    Diatas terdapat data x = 3 dan y = 5,maka data tersebut termasuk kategori apa ?Bad atau Good.

    Langkah-Langkah penyelesaian :

    1. Kita tentukan parameter K, misalkan K=3

    2. Hitung jarak data baru dengan semua data training menggunakan rumus euclidean distance

    1. Kita urutkan jarak dari data baru dengan data training dan menentukan tetangga terdekat berdasarkan jarak minimum K.

    1. Tentukan kategori dari tetangga terdekat. Kita perhatikan baris 3, 4, dan 5 pada gambar sebelumnya (diatas). Kategori Ya diambil jika nilai K<=3. Jadi baris 3, 4, dan 5 termasuk kategori Ya dan sisanya Tidak.

    Kategori ya untuk K-NN pada kolom 6, mencakup baris 3,4, dan 5. Kita berikan kategori berdasarkan tabel awal. baris 3 memiliki kategori Bad, dan 4,5 memiliki kategori Good.

    1. Gunakan kategori mayoritas yang sederhana dari tetangga yang terdekat tersebut sebagai nilai prediksi data yang baru.

    Program KNN Menggunakan Data Iris¶

    Import pandas (pandas untuk membersihkan data mentah ke dalam sebuah bentuk yang cocok untuk analisis yaitu tabel ) dan sklearn ( Scikit-Learn memberikan sejumlah fitur untuk keperluan data science seperti: Algoritma Regresi, Algoritma Naive Bayes, Algoritma Clustering,dll )

    from sklearn import datasets
import pandas as pd
    from sklearn.linear_model import logistic_regression_path

    memasukkan data iris/import data

    iris=datasets.load_iris()

print(iris.data)
print(iris.target)

    Membagi data dan melatih model

    Membagi data menjadi data training dan data testing yakni dengan menggunakan train_test_split.

    from sklearn.model_selection import train_test_split
x_train, x_test, y_train, y_test=train_test_split(iris.data,iris.target,test_size=0.33)

    Selanjutnya kita impor kelas KNeighborClassifier 

    from sklearn.neighbors import KNeighborsClassifier
clf=KNeighborsClassifier(n_neighbors=3).fit(x_train,y_train)
    from sklearn.metrics import accuracy_score
print("accuracy is ")
print(accuracy_score(y_test,clf.predict(x_test)))

    Import Matplotlib untuk membantu memvisualisasikan data dengan indah dan rapi sehingga kita dapat menampilkan data yang telah kita olah sesuai dengan kebutuhan.

    
import matplotlib.pyplot as plt

    
accuracy_values=[]

    Metode fit digunakan untuk melatih model tentang data pelatihan(x_train,y_train) dan metode 'predict' untuk melakukan pengujian pada data pengujian(x_test). memilih nilai optimal K sangat penting,jadi kami menyesuaikan dan menguji model untuk nilai yang berbeda untuk K menggunakan LOOP untuk mencatat akurasi pengujian KNN dalam sebuah sebuah variabel.

    for x in range(1,x_train.shape[0]):
    clf=KNeighborsClassifier(n_neighbors=x).fit(x_train,y_train)
    accuracy=accuracy_score(y_test,clf.predict(x_test))
    accuracy_values.append([x,accuracy])
    pass

    import numpy( untuk operasi vektor dan matriks. Fiturnya hampir sama dengan MATLAB dalam mengelola array dan array multidimensi.)

    import numpy as np
accuracy_values=np.array(accuracy_values)

    Plot hubungan antara nilai-nilai K dan akurasi pengujian yang sesuai menggunakan perpustakaan matplotlib.

    secara umum,akurasi pelatihan meningkat seiring dengan meningkatnya kompleksitas model,untuk KNN kompleksitas model ditentukan oleh nilai K. nilai K yang lebih besar mengarah ke batas keputusan yang lebih halus(model yang kurang kompleks). K yang lebih kecil mengarah ke model yang lebih kompleks. Kita mendapatkan akurasi pengujian maksimum ketika model memiliki tingkat kompleksitas yang tepat, dengan itu dengan menggunakan modul ini di dapat akurasi sebesar 0,94

    plt.plot(accuracy_values[:,0],accuracy_values[:,1])
plt.xlabel("K")
plt.ylabel("accuracy")
plt.show()
